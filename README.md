# 블로그 프로젝트

## 기술스택
- JDK 11
- Springboot 2.7.8
- Mybatis 
- 테스트 h2 DB
- 프로덕션 MySQL DB
- JSP

## 모델링
### 1단계 
- User
- Board
### 2단계
- Reply
### 3단계
- Love

## 기능정리
### 1단계 (완료)
- 회원가입
- 로그인
- 글쓰기
- 글 목록보기 (썸네일 제외)
- 글 상세보기
- 글 수정 및 삭제
- 썸네일 보기

- 댓글 쓰기
- 댓글 목록 보기
- 댓글 삭제하기
- 프로필 사진 추가하기

- 아이디 중복 체크
